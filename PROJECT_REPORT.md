## 4. Key Architectural Decisions

1.  **Glassmorphism vs. Performance:** Opted for high-blur backdrops (`backdrop-blur-xl`) but mitigated the GPU cost by using `will-change-transform` and lazy-loading heavy sections.
2.  **Hybrid Motion:** Used `motion/react` for layout transitions and `GSAP` for high-frequency micro-interactions (like the confetti and early bird pulses).
3.  **Proxy Routing:** Used Nginx to proxy API requests, allowing the landing page to remain a pure static build while handling dynamic lead data securely.
